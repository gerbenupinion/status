# AML Simple Status

Public status page for [AML Simple](https://amlsimple.com) — powered by [Upptime](https://github.com/upptime/upptime).

**Live:** [status.amlsimple.com](https://status.amlsimple.com)

## Monitored Services

| Service | URL |
|---------|-----|
| Landing Site | https://amlsimple.com |
| App | https://app.amlsimple.com |
| Auth | https://auth.amlsimple.com |
| API / Edge Functions | https://api.amlsimple.com |
| Database | https://db.amlsimple.com |

## How It Works

- GitHub Actions checks each endpoint every 5 minutes
- Response time data collected daily
- Status page auto-deployed to GitHub Pages
- Incidents auto-created as GitHub Issues
